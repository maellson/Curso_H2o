Importar a Imagem em Outro Computador:

1. Favor usar o seguinte comando para importar a imagem e criar um contêiner
*docker load: Carrega uma imagem a partir de um arquivo .tar.
* -i desafio-db-export.tar: O arquivo de imagem que fornecido.

--codigo--  

docker load -i desafio-db-export.tar

------------



2. Executar o Contêiner:

Depois de carregar a imagem, voces podem executar o contêiner:

--codigo--  

 docker run --name desafio-container -p 5434:5432 -d desafio-db-export

-----------
